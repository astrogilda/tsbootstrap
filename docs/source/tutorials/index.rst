Tutorials
=========

Runnable notebooks that teach the library by example. Each one renders here with
its saved outputs and can be launched interactively on Colab or Binder using the
badges at the top of the notebook. Start with the getting-started pair, then use
the decision guide to choose a method; the remaining sections go deeper on each
method family and on uncertainty quantification.
